# Security

## Scope

This repository contains a static frontend for Centre D'Auto Allard. There is no backend, database, authentication layer, or private API surface in this project.

## Secrets

Do not commit secrets, private keys, deployment credentials, API tokens, or production environment files. Public frontend configuration belongs in `.env.example`.

## Reporting

Report security concerns privately to the repository owner. Include the affected path, expected behavior, observed behavior, and reproduction steps where possible.

## Supported Versions

Only the current production branch is supported.
